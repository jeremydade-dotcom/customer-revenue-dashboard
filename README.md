# Customer Revenue & Performance Dashboard

## Project Overview
Interactive Tableau dashboard analyzing customer revenue trends, business performance, and key profitability metrics. This project focuses on identifying revenue drivers, customer performance patterns, and high-level business insights through data visualization and KPI analysis.

## Tools Used
- Tableau
- SQL
- Excel

## Key Insights
- Identified top-performing customer segments
- Analyzed revenue and profit trends
- Compared business performance across categories
- Highlighted customer contribution to overall revenue

## Dashboard Features
- Interactive filters
- KPI scorecards
- Revenue trend analysis
- Customer performance breakdowns
- Profitability analysis visuals
